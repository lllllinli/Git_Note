#Git基本指令#
<br>
## 1. 在本機建立一個 repository ##

`````
/*
* 首次建立 Git 專案
*
* 1.移動路徑到'專案資料夾'。
* 2.建立一個Git_Test專案目錄。
* 3.移動到'專案資料夾'下。
* 4.產生一個新的 repository。
*/

$cd 資夾路徑 
$mkdir 專案名稱
$cd 專案名稱
$git init


`````

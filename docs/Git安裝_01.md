##### 2. 設定 Git config #####

---------------------

必須設定三處：
<br><br>
**1.'system'設定：**
<br><br>
檔案 **`/etc/gitconfig`**： 包含給該系統所有使用者的儲存庫使用的數值。 只要讀者傳遞 --system 參數給 git config，它就會讀取或者寫入參數到這個檔案。
<br><br>
**2.'使用者'設定：**
<br><br>
檔案 **`~/.gitconfig`**： 給讀者自己的帳號使用。 傳遞 --global 參數給 git config，它就會讀取或者寫入參數到這個檔案。
